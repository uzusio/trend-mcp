# ニュース選択のフィードバック

直前の `/post-trend` の結果に対してフィードバックを行い、傾向ファイルを更新します。

## 手順

1. ユーザーに以下を確認:
   - 直前に選ばれたニュースは良かったか？（良い / 悪い）
   - その理由は？（任意）

2. `config/preferences.md` を読み込む

3. フィードバック内容に基づいて傾向ファイルを更新:
   - **良い場合**: 「興味のある傾向」セクションに、なぜ良かったかの要素を追加
   - **悪い場合**: 「興味のない傾向」セクションに、なぜ悪かったかの要素を追加

4. 「フィードバック履歴」セクションに記録を追加:
   ```
   - [日付] ニュース: 「タイトル」 → 良い/悪い（理由）
   ```

5. 更新した傾向ファイルを保存

## 例

ユーザー: 「悪い。スポーツの結果には興味ない」

→ 「興味のない傾向」に「スポーツの試合結果」を追加

ユーザー: 「良い！新作ゲームの情報は嬉しい」

→ 「興味のある傾向」に「ゲームの新作発表・続報」を追加
