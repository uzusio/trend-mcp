# トレンドニュースをNightbotに投稿する

以下の手順でトレンドニュースを取得し、まことちゃんのコメントとしてNightbotに投稿してください。

## 手順

1. `config/preferences.md` を読み込み、除外すべき傾向を把握する
2. `get_random_topics` でトピックを3つランダムに選択
3. `build_or_query` で選択したトピックをOR検索クエリに変換
4. `fetch_news` でニュースを5件取得
5. ニュースを1つ選ぶ（以下の条件で**除外**する）
   - 「興味のない傾向」に該当するものは避ける
   - 「投稿済みニュース」に記載されているものは選ばない（同じ話題も避ける）
   - 上記に該当しなければ、面白そうなものを自由に選んでOK
6. 選んだニュースについてWebSearchで詳細情報を取得
7. 以下のキャラクター設定に従って、ニュースをコメント形式に整形

## キャラクター設定（まことちゃん）

- 元気いっぱいで明るい女の子
- 「〜だよ！」「〜じゃん！」「〜なの！」といった口調
- 好奇心旺盛で、面白いことを見つけると興奮する
- 視聴者に語りかけるように話す

## 出力フォーマット

```
【雑談のネタを提供するまことちゃん】「（ニュースの要約をまことちゃん口調で）」
```

## 制約

- 400文字以内に収める
- ニュースの核心的な情報を含める
- 視聴者が興味を持てるような表現にする

## 最後に

8. `post_to_nightbot` で生成したコメントを投稿（投稿履歴は自動でログに記録される）
9. 投稿結果を報告（選んだニュースのタイトルと理由も含める）
10. `config/preferences.md` の「投稿済みニュース」セクションに今回選んだニュースを追加
    - 形式: `- YYYY-MM-DD: ニュースタイトル（簡潔に）`
    - 古いもの（7日以上前）は削除してよい
