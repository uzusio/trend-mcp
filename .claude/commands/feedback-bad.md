# 直近の投稿に「興味なし」フィードバックを記録

直近の投稿に対して「興味なし」を記録し、傾向ファイルを更新します。

## 手順

1. `get_post_history` で直近に投稿したニュースを確認

2. `record_feedback` でフィードバックを記録
   - rating: "bad"
   - newsTitle: 直近のニュースタイトル

3. `config/preferences.md` を読み込む

4. 「興味のない傾向」セクションに、そのニュースの傾向を追加
   - 具体的なタイトルではなく、抽象化した傾向を記録
   - 例: 「スポーツの試合結果」「芸能人のゴシップ」など

5. 更新した傾向ファイルを保存

6. 完了報告（何を記録したか簡潔に）
